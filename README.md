# 📊 数学函数可视化工具

一个基于 Python 和 Tkinter 的数学函数可视化工具，支持多种函数类型的绘制和分析。

## ✨ 主要特性

### 🎯 核心功能
- **多函数类型支持**: 二次函数、三角函数、指数函数、对数函数
- **实时参数调节**: 通过输入框调整函数参数
- **多函数叠加**: 支持在同一图形中显示多个函数
- **特征点标注**: 自动标注极值点、零点、交点等关键特征
- **图像保存**: 支持将图形保存为高质量PNG图片

### 🎨 界面特色
- **现代化设计**: 清新的界面风格和配色方案
- **固定布局**: 稳定的左右分栏布局，确保绘图区域始终可见
- **图标化操作**: 直观的图标按钮设计
- **中文支持**: 完整的中文界面和字体支持

### 🔧 技术特性
- **字体管理**: 智能中文字体检测和设置
- **参数验证**: 智能的函数参数有效性检查
- **错误处理**: 完善的异常处理和用户提示
- **模块化设计**: 清晰的代码结构，易于维护和扩展

## 🚀 快速开始

### 环境要求
- Python 3.7+
- tkinter (通常随Python安装)
- matplotlib
- numpy

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

## 📖 使用说明

### 基本操作
1. **选择函数类型**: 从下拉菜单选择要绘制的函数类型
2. **设置参数**: 在参数输入框中输入函数的系数 a、b、c
3. **绘制函数**: 点击"📊 绘制函数"按钮查看图形
4. **添加函数**: 使用"➕ 添加函数"在同一图形中叠加多个函数
5. **清除图形**: 点击"🗑️ 清除图形"清空所有函数
6. **保存图像**: 点击"💾 保存图像"将当前图形保存为PNG文件
7. **字体设置**: 点击"🔤 字体设置"打开字体管理窗口

### 支持的函数类型
- **二次函数**: 📐 y = a·x² + b·x + c
- **正弦函数**: 〰️ y = a·sin(b·x + c)
- **余弦函数**: 〰️ y = a·cos(b·x + c)
- **正切函数**: 📈 y = a·tan(b·x + c)
- **指数函数**: 📊 y = a·e^(b·x) + c
- **对数函数**: 📉 y = a·log(b·x + c)

### 特征点显示
- **极值点**: 函数的最大值和最小值点
- **零点**: 函数与x轴的交点
- **交点**: 多个函数之间的交点（添加多个函数时显示）

## 🎨 字体设置

### 字体管理功能
- **自动检测**: 自动检测系统中可用的中文字体
- **字体预览**: 实时预览字体效果
- **字体应用**: 一键应用选择的字体到图形
- **字体信息**: 在控制面板显示当前使用的字体信息

### 推荐字体
- **Windows**: 微软雅黑、宋体、黑体
- **macOS**: PingFang SC、Heiti SC
- **Linux**: Noto Sans CJK、WenQuanYi

## 🎯 使用指南

### 基本操作

1. **选择函数类型**: 在下拉菜单中选择要绘制的函数
2. **调整参数**: 修改a、b、c参数值
3. **绘制函数**: 点击"📊 绘制函数"按钮
4. **添加函数**: 点击"➕ 添加函数"进行多函数对比
5. **保存图像**: 点击"💾 保存图像"导出结果

### 字体设置

1. 点击"🔤 字体设置"按钮
2. 在字体列表中选择喜欢的字体
3. 在预览区域查看效果
4. 点击"应用字体"按钮
5. 图形中的中文标签将使用新字体

### 参数说明

- **a参数**: 影响函数的幅度或缩放
- **b参数**: 影响函数的频率或斜率
- **c参数**: 影响函数的相位或偏移

## 🐛 故障排除

### 常见问题

1. **启动失败**: 检查Python环境和依赖包
2. **字体显示异常**: 确认选择的字体支持中文
3. **图形不显示**: 检查matplotlib安装
4. **参数错误**: 确认输入的参数值合理

### 解决方案

```bash
# 重新安装依赖
pip install --upgrade matplotlib numpy

# 检查Python版本
python --version

# 查看错误信息
python main.py
```
